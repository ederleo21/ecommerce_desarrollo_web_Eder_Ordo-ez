# 🌟 Aura Store

Tienda de electrónica minimalista con diseño premium, construida con HTML, CSS y JavaScript vainilla. Incluye catálogo dinámico, carrito de compras y pasarela de pago simulada.

## 🚀 Ejecución Local

Para ejecutar el proyecto en tu computadora:
1. Abre una terminal en la carpeta del proyecto.
2. Ejecuta: `npx serve ./`
3. Abre `http://localhost:3000` en tu navegador.
